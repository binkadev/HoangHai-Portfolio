# Binkadev Cinematic Portfolio

A Vue 3 + Vite + TypeScript personal portfolio for Tran Hoang Hai (`binkadev`).

The design direction is a modern cinematic portfolio with soft brush-inspired motion, a moving `THH` logo, deep gradients, project cards, notes, toolkit, and contact sections.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Recommended: Vercel or Netlify.
